## 1. 核心技术栈

| 维度 | 技术选型 | 说明 |
| --- | --- | --- |
| **开发模式** | 前后端分离 | 严格解耦，通过 RESTful API 进行交互 |
| **后端语言** | **Python** | 需遵循 PEP 8 代码规范 |
| **后端框架** | **FastAPI** | 必须利用其异步特性（Async/Await）与类型提示 |
| **前端框架** | **React** | 结合 **Vite** 构建工具，确保极速热更新 |
| **前端语言** | **JavaScript** | 保持代码简洁，优先使用 ES6+ 语法 |
| **数据库** | **SQLite** | 轻量级关系型数据库，确保符合 ACID 事务标准 |
